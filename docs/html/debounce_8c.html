Get and consume a pending falling edge event.

A falling edge event is generated when a valid high to low transition, has been detected by the debounce filter

Initialize a debounce filter instance.

Configures a debounce filter and resets its internal state

The filter will begin collecting samples on subsequent calls to e_debounce_update()

Parameters
ps_dbPointer to the debounce instance to initialize
ui32_idApplication defined identifier associated with the instance
readRaw input read callback. Must not be NULL
contextUser context pointer passed to the read callback. May be NULL if not required
ui8_cyclesNumber of consecutive identical samples required to validate a state transition. Values are clamped to the range [1, 31]
Returns
Debounce operation status
